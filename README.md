## 주제   
운동 동작 인식 알고리즘 개발   
스마트 헬스케어 산업에 적용 가능한 데이터 분석 방법 
## 배경   
3축 가속도계(accelerometer)와 3축 자이로스코프(gyroscope)를 활용해 측정된 센서 데이터에 머신러닝 알고리즘을 적용해 운동 동작 인식 알고리즘 개발
